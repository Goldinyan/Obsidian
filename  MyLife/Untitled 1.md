
Der Ereignishorizont: Die Grenze ohne Wiederkehr

Definition: 

-> äußere Grenze eines SL
-> keine feste Oberfläche (wie bei Planeten)
-> sondern mathematische Markierung im Raum

Point of No Return:

-> schon kurz vorhin besprochen
-> Grenze 
-> bei Überschreitung kein physikalischen Weg zurück

Anziehungskraft:

-> Gravitation so stark
-> muss schneller als Licht sein zum entkommen
-> unmöglich -> alles bleibt gefangen

Optik:

-> Beobachter von außen sieht erstaunliches
-> Objekt fällt nicht rein in den Ereignishorizont 
-> sondern bleibt hängen 
-> wird immer röter und schwächer


Berechnung des Schwarzschildradius:

-> Radius des Ereignishorizonts nennt man Schwarzschild-Radius
-> berechnet von Karl Schwarzschild (Deutscher) im Jahr 1915
-> 1 Jahr nach der Allgemeinen Relativitätstheorie von Albert Einstein

$$r_s = \frac{2GM}{c^2}$$

-> M ist Masse
-> G die Gravitationskonstante
-> c die Lichtgeschwindigkeit
-> Ergebnis ist in Metern
