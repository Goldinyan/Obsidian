
# 1. Häufig genutzte Keybinds (90 % Alltag)

## Fensterfokus (Navigation)
- `lalt - h/j/k/l` → Fokus links / unten / oben / rechts  
- `lalt - r` → letztes Fenster  
- `lalt - m` → Fullscreen‑Fenster durchschalten  

## Fenster verschieben (Swap)
- `alt + shift - h/j/k/l` → Fenster nach links / unten / oben / rechts tauschen  

## Workspace wechseln
- `lalt - 1…0` → Space 1–10  
- `lalt - 0x21` → vorheriger Space  
- `lalt - 0x1E` → nächster Space  

## Fenster auf anderen Workspace verschieben
- `alt + shift - 1…0` → Fenster → Space 1–10  
- `alt + shift - 0x21` → Fenster → vorheriger Space  
- `alt + shift - 0x1E` → Fenster → nächster Space  

## Floating / Fullscreen / Split
- `meh - t` → toggle floating  
- `meh - m` → toggle fullscreen  
- `meh - s` → toggle split  

## Terminal öffnen
- `cmd - return` → kitty

# 2. Gelegentlich genutzte Keybinds (Layout, Displays, Spaces)

## Displays wechseln
- `lalt - u/i/o` → Fokus Display links / Mitte / rechts  
- `lalt - z/x` → Fokus Display west/east (wrap)  

## Fenster auf anderes Display verschieben
- `alt + shift - u/i/o` → Fenster → Display links / Mitte / rechts  
- `alt + shift - z/x` → Fenster → Display west/east (wrap)  

## Space‑Management
- `alt + ctrl - n` → neuen Space erstellen  
- `alt + ctrl - w` → Space löschen  
- `alt + ctrl - 1…0` → Space an Position verschieben  
- `alt + ctrl - 0x21/0x1E` → Space nach links/rechts verschieben  

## Layout wechseln
- `alt + ctrl - b` → bsp  
- `alt + ctrl - f` → float  
- `alt + ctrl - s` → stack  

## Resize (Tiling)
- `fn - j/h/k/l` → Resize links/rechts/oben/unten  
- `fn - 0` → Balance

# 3. Selten genutzte Keybinds (Spezialfälle, Power‑User)

## Stack‑Navigation
- `lalt - 0x2B` → stack.prev  
- `lalt - 0x2F` → stack.next  
- `lalt - 0x2C` → stack.recent  

## Warp (Fenster neu einordnen)
- `alt + shift - w/s/a/d` → warp north/south/west/east  

## Insert‑Position
- `alt + shift - up/down/left/right` → insert north/south/west/east  
- `alt + shift - 0x2C` → insert stack  

## Floating‑Fenster bewegen
- `fn + shift - h/j/k/l` → move floating window  

## Floating‑Grid‑Splits
- `fn + shift - w/s/a/d` → obere/untere/linke/rechte Hälfte  
- `fn + shift - m` → fullscreen grid  
- `fn + shift - 0` → restore centered  

## Space erstellen + Fenster verschieben
- `alt + shift - n` → neuen Space erstellen + Fenster dorthin + Fokus  

## System
- `hyper - r` → yabai neu starten  
- `hyper - x` → passthrough mode
