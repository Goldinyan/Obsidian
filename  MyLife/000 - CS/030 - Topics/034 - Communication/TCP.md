# TCP

TCP also known as **Transmission Control Protocol** is a connection-oriented transport layer protocol of the Internet Protocol suite that ensures reliable, delivery of data between applications by establishing a connection, managing data transmission, error checking, and recovery, typically used to support application-layer protocols like HTTP.
