### Installation and Usagwe

w3m is a chromium based browser for your terminal and can be installed with `brew install w3m`.

The application itself is pretty easy to use, with the following commands:

To start it type `w3m` followed by the domain, e.g. `w3m https://www.wikipedia.org`.

### Navigation

To navigate use basic vim key binds (h, j, k, l) 

Tab         -> jumps to next link
Shift + Tab -> jumps to the previous link
Space       -> scrolls one page down 
b           -> scrolls one page up 
g           -> goes to start of the page
G           -> goes to bottom of the page


### Links and Browser-History

Enter       -> opens current link
B           -> goes one page back in the history
v           -> shows HTML code
s           -> shows history 

### Tabs 

Shift + T   -> current link in new tab  
Shift + U   -> allows new url for new tab
{           -> switching trough tabs
}           -> switching trough tabs
Ctrl + q    -> closes current tab


### Searching on a site

 /          -> searching a word from the top
 ?          -> searching a word from the bottom
 n          -> jumping to next hit 
 N          -> jumping to previous hit 


### Bookmarks

Shift + A   -> adds site to bookmark
Shift + B   -> opens bookmark list
